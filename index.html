Magic Square (C Implementation)

This magic square implementation creates a magic square by starting in the center of the first row. It then iterates through to n2 until the matrix is full. It moves one space up and one to the right wrapping around at the edges. If the spot is not empty, it iterates one row down. You can try tracing these steps by inputting a small value above and tracing this pattern.

Sudoku Board Checker

In this project we created a C program that would validate a text file to see if it contained a valid sudoku square. The text function required the size of the board followed by a newline and then the board. The board was comma seperated values from 0-9, 0 defining a spot as empty. Our program would then parse through the text file and check if the board that was given was valid. I have included a few screenshots of my own code completed for this project.

This main method first checks that the arguments passed to this executable are correct, then proceeds to create a 2D array reflecting the size of the board. It then reads the board passed as input to fill in the 2D array before passing it to the checker method shown below.

This validator method first creates 0 new arrays for each iteration since it checks the column and the row simutaneously. It first checks that it contains a valid number. It then checks that the same index is empty in the designated column/row array. If it is not empty, it means we have found two of the same number in the same row/column. This project also required us to use array arithmetic instead of indexing.
